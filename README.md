## Sobre o Projeto

O ObservaAção é um sistema desenvolvido para facilitar a comunicação entre cidadãos e órgãos públicos, permitindo o registro e acompanhamento de solicitações e denúncias de forma simples, segura e eficiente.


## Objetivo

Criar uma plataforma acessível para diferentes perfis de usuários, garantindo:

* Facilidade de uso
* Segurança das informações
* Organização no atendimento das demandas


## Perfis de Usuários

### Cidadão com baixa familiaridade digital

Usuários com pouca experiência tecnológica. O sistema foi pensado com:

* Interface simples
* Botões maiores
* Linguagem clara
* Feedback visual para orientar ações


### Cidadão em situação de vulnerabilidade

Usuários que podem ter receio de se expor ao realizar denúncias. O sistema oferece:

* Possibilidade de anonimato
* Segurança e privacidade
* Linguagem simples e acolhedora
* Confirmação e acompanhamento da solicitação


### Servidor Público / Gestor

Usuários responsáveis por gerenciar as solicitações. O sistema oferece:

* Interface organizada
* Filtros e relatórios
* Acompanhamento de demandas
* Agilidade no atendimento


## Aplicação de Clean Code

O projeto aplicou boas práticas para melhorar a qualidade e manutenção do código.

### Método criar (SolicitacaoService)

* Responsabilidade única
* Métodos pequenos
* Nomeação clara

Benefício: código mais organizado e fácil de manter.

### Método atualizarStatus (SolicitacaoService)

* Validação de dados
* Tratamento de erros
* Código defensivo

Benefício: maior confiabilidade e menos erros.


### Método buscarPorProtocolo (SolicitacaoRepository)

* Código mais simples
* Uso de Stream API
* Melhor legibilidade

Benefício: código mais fácil de entender e manter.
